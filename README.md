# ARM Boot, Virtual Memory & Execution Modes

A hands-on, architecture-focused repository that explains **how ARM systems boot**, **how pageable virtual memory works**, and **how AArch32 and AArch64 differ**, using **clear explanations, diagrams, QEMU experiments, and real SoC examples (e.g. i.MX6)**.

This repo is meant to bridge the gap between *ARM Architecture manuals* and *real Linux/embedded bring-up work*.
